# ファイル自動仕分け・リネームツール

指定フォルダ内のファイルを拡張子や日付で自動仕分け・リネームするツールです。

## 機能
- 拡張子別にフォルダを自動作成・仕分け
- 日付別にフォルダを自動作成・仕分け
- ファイルの一括リネーム
- 処理ログの自動出力

## 使用技術
- Python 3.x
- os
- shutil
- pathlib

## 使い方
1. リポジトリをクローン
2. スクリプトを実行
```bash
python main.py
```

## 出力例
- `output/画像/` - 画像ファイル(.jpg, .png等)
- `output/文書/` - 文書ファイル(.pdf, .docx等)
- `output/動画/` - 動画ファイル(.mp4等)
- `output/その他/` - その他のファイル
- `output/log.txt` - 処理ログ
